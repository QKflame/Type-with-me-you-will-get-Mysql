# 五、MySQL 服务的登录和退出

**登录：**

```mysql
mysql -h主机名 -p端口号 -u用户名 -p密码
```

**注意：如果是本机且端口号是3306，则 ``-h主机名 -p端口号`` 可以省略。**

**-h和主机名之间、-p和端口号之间、-u和用户名可以加空格，-p和密码之间是不能加空格的。**



**退出：**

exit 或 ctrl+c
